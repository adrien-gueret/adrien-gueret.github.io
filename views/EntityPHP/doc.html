EntityPHP adrien-gueret/EntityPHP

A basic and easy to use ORM built with and for PHP.

1. Include EntityPHP in your project

require 'path/to/EntityPHP.php';

2. Create a class inherited from \EntityPHP\Entity

class Characters extends \EntityPHP\Entity
{
	protected $firstname;
	protected $age;

	public static function __structure()
	{
		return array(
			'firstname'	=>	'VARCHAR(255)',
			'age'		=>	'INT(11)',
		);
	}
}

3. Connect to your database and generate your tables

\EntityPHP\Core::connectToDB('hostname', 'username', 'password', 'databasename');
\EntityPHP\Core::generateDatabase();

4. Create and save a new instance of your class

$character	=	new Characters();
$character->setProps(array(
	'firstname'	=>	'John',
	'age'		=>	20,
));
Characters::add($character);

5. That's it! Your table is generated and populated!
